Expense Tracker Dashboard

A web-based expense tracker built using HTML, CSS, and JavaScript. It allows users to track income and expenses, view summaries, and analyze spending with charts.

Features

Add income and expenses
Categorize transactions
View total income, expenses, and net balance
Charts for income vs expenses and category-wise spending
Filter and sort transactions
Delete or clear data
Export data as JSON
Data stored using browser localStorage

Technologies

HTML
CSS
JavaScript
Chart.js

How to Run

Open index.html in any modern web browser. No setup required.

Storage

All data is saved locally in the browser using localStorage.
